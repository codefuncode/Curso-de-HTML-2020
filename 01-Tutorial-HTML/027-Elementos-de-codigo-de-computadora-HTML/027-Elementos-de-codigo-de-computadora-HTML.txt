Resumen del capítulo

	El elemento kbd se usa para definir la entrada del teclado

	El elemento de muestra se usa para definir la salida de muestra de un programa de computadora

	El elemento de código se usa para definir un fragmento de código de computadora

	El elemento var se usa para definir una variable en la programación o en una expresión matemática

	El elemento pre se usa para definir texto preformateado
	
============================================================
Chapter Summary

	The <kbd> element is used to define keyboard input

	The <samp> element is used to define sample output from a computer program

	The <code> element  is used to define a piece of computer code

	The <var> element  is used to defines a variable in programming
	or in a mathematical expression

	The <pre> element is used to define preformatted text
