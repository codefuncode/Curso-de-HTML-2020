• Semántica:

    Te permite describir con mayor precisión cuál es tu contenido. Lo que permite a los navegadores comprender tu contenido y mostrar mejores resultados de búsquedas.

  • Conectividad:

    Permite comunicarse con el servidor de formas nuevas e innovadoras.

  • Sin conexión y almacenamiento:

    Permite que las páginas web almacenen datos en el lado del cliente localmente y operen sin conexión de manera más eficiente.

  • Multimedia:

    Trabajar con vídeo y audio en la Web Abierta.

  • Gráficos y efectos 2D / 3D:

    Permiten una gama mucho más diversa de opciones de presentación.

  • Rendimiento e integración:

    Proporciona una mayor optimización de la velocidad y un mejor uso del hardware informático

  • Acceso al dispositivo:

    Permite el uso de varios dispositivos de entrada y salida.

  • Estilo:

    Permitir que los autores escriban temas más sofisticados.

  • Secciones y esquemas en HTML5

    Un vistazo a los nuevos elementos de esquema y sección en HTML5: sección, artículo, navegación, encabezado, pie de página y aparte.

  • Usar audio y vídeo HTML5

    Los elementos de audio y vídeo se integran y permiten la manipulación de nuevos contenidos multimedia.

  • Mejoras de formularios

    Una mirada a la API de validación de restricciones, varios atributos nuevos, nuevos valores para el tipo de atributo de entrada y el nuevo elemento de salida.

  • Nuevos elementos semánticos

    Además de las secciones, los medios y los elementos de formularios, hay numerosos elementos nuevos, como marca, figura, título de figura, datos, tiempo, salida, progreso o medidor y principal, lo que aumenta el número de elementos HTML5 válidos.

  • Mejora en iframe

    Usando los atributos sandbox y srcdoc, los autores ahora pueden ser precisos sobre el nivel de seguridad y la representación deseada de un elemento iframe.

  • MathML

    Permite incrustar directamente fórmulas matemáticas.
